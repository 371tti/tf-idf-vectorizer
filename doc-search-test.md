# doc-search-test

`vectorizer.search()` メソッドで論理式 + cos類似度を使用した検索テストです  
queryでは`&` `|` `!` `[]` が使えるようにしてます。

## index構築
sudachi termizerを用い、C modeでtermizeしたのち それらの語彙をさらにA modeでtermizeしています。
